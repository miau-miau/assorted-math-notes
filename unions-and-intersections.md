#### Intersections

>[!definition]
>The **intersection** of $A$ and $B$ 

```tikz
\begin{document}

\begin{tikzpicture}[thick,
	set/.style = {circle,
		minimum size = 3cm,
		fill=pink,
		opacity=0.5}]

% Set A
\node[set,label={135:$A$}] (A) at (0,0) {};

% Set B
\node[set,label={45:$B$}] (B) at (1.8,0) {};

% Set intersection label
\node at (0.9,0) {$A\cap B$};

\end{tikzpicture}

\end{document}

```
